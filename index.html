H2 database.

Compliant

Fully respects the OGC's Simple Features for SQL standards.

Solid

Built on the JDBC compliant H2 database.

Tested

Passes over 559 unit tests (and counting).

Getting started
couldn't be simpler.

Quick-start Instructions

# Download, unzip & run jar

~ $ wget http://goo.gl/cdfi7b -O h2gis.zip

~ $ unzip h2gis.zip

~ $ cd h2gis-standalone

~/h2gis-standalone $ java -jar h2-dist-1.1.0.jar

# => Click Connect in the web interface
