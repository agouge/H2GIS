### Signature

{% highlight mysql %}
DOUBLE ST_XMin(GEOMETRY geom);
{% endhighlight %}

### Description

Returns the minimum x-value of `geom`.

### Example

{% highlight mysql %}
SELECT ST_XMin('LINESTRING(1 2 3, 4 5 6)');
-- Answer:    1.0
{% endhighlight %}
